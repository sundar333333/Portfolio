I want to implement a cinematic "Scrollytelling" transition from the Landing Page to the Projects Gallery.

This involves a specific sequence of camera movements and scene changes triggered by the `ScrollControls`.

### Phase 1: The "Dive In" Transition (Scroll 0% - 15%)
When the user begins scrolling down from the top:
1.  **Audio/Video Cut:** If the "Messi Tribute" video is playing, immediately stop it. Force the TV screen texture back to the "No Signal" static noise.
2.  **Camera Zoom:** The camera should move forward rapidly, zooming directly **into** the TV screen.
3.  **The Wipe:** The movement should continue until the "No Signal" static noise fills the entire viewport (100% width/height). This should look identical to the initial Loading Screen (but without the countdown text). This acts as a visual "wipe" or portal.

### Phase 2: The Scene Shift (Scroll 15% - 20%)
Once the screen is fully filled with static, seamlessy transition to the "Project Gallery" state:
1.  **Environment Change:** Remove/Hide the 3D Room floor, walls, and the Gyroscope mouse effect. The TV should now be floating in a void.
2.  **TV Position:** The TV zooms back out, but now it is locked in the **dead center** of the screen, facing the user directly (flat 2D look).
3.  **Background:** The background is no longer black. It should be a **Fullscreen Plane** behind the TV displaying a heavily **Pixelated/Mosaic version** of the current project's poster.

### Phase 3: The Project Slider (Scroll 20% - 100%)
Now the user is in the "Gallery Mode" (Reference: alche.studio scroll feel).
1.  **The TV Content:** As the user scrolls, the image on the TV screen slides (parallax effect) to show the next Project Poster.
2.  **The Background:** Synced with the TV change, the pixelated background smoothly transitions colors/shapes to match the new project.
3.  **Layout:**
    - **Center:** The TV showing the clear project image.
    - **Bottom Left:** Project Title & Description text (HTML overlay).
    - **Bottom Right:** "View Case Study" button.
    - **End of Scroll:** A "+" slide for "Add Your Project".

**Technical Requirement:**
Use `useScroll` from `@react-three/drei` to map these ranges. Use a shader or a low-resolution render target for the pixelated background effect.
